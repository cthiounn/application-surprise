About this website

It was done in an educational purpose for the Cloud computing course of ENSAE by Beatriz Farah, Lucie Tournier, Fyras Telmini and Conrad Thiounn. It is the final product of hundreds of hours of hard work, in data/ML/devops.

Actor-Director graph was done by Beatriz using neo4j. The resulting image can be zoomed with the mouse.

Interactive Actor-Director graph was also done by Beatriz using Plotly and is another version, but interactive.

Music 30 Days Challenge Map was done by Conrad using Folium and Wikipedia. It is showcasing a map of countries where the best sold singles artists came from.

Pokemon generator was done by Fyras using Stable diffusion and Rembg and Streamlit. It takes a picture and a prompt and turn into a pokemon image.

All these projects was supervised and directed by Lucie.
